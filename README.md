# 基于ESP32最小系统的便携钥匙扣开发板

## 硬件架构
### MUC
- 采用ESP32-S3FH4R2作为主控单元
- 全引脚通过2.54mm排针引出

### 电源管理
- 搭载ME6217C33M5G低压差线性稳压器，支持2-6.5V宽电压输入
- 输出3.3V/800mA稳定供电

### 系统配置
- BOOT/RST引脚预置4.7kΩ上拉电阻
- 集成贴片陶瓷天线
- 采用0402、0603(10uF)封装阻容器件,板框尺寸31.0mm×31.0mm


| 项目         | 规格              |
|--------------|-------------------|
| 工作电压     | 2.0-6.5V DC       |
| 最大功耗     | 850mW@5V输入      |
